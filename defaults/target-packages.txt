# Default packages cross-emerged into every board's target sysroot.
#
# Per-board overrides in boards/<board>/target-packages.txt:
#   foo/bar        add to the set
#   -foo/bar       remove from the default set
#
# Mirrors the practical default lifted from cross-stage.sh
# (ADDITIONAL_PACKAGES + install_boot), trimmed of language toolchains
# that aren't universally wanted (rust/go/mold — opt in per board).
